## dotfiles

*各地からコピペの嵐*


* tmux
    * tmux-sensible
    * tmux-resurrect
    * tmux-sidebar

* zsh
    * zaw
    * zsh-syntax-highlighting

* vim
    * incsearch
    * neocomplcache
    * Unite
    * VimFiler
    * scala

* powerline

* git


## Usage

```shell
git clone git@github.com:amaya-382/dotfiles.git dotfiles
cd dotfiles
./init.sh
```
